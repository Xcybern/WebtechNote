什么是 CSS（Cascading Style Sheets）

CSS 告诉浏览器：内容应该如何呈现（颜色、字体、间距、布局、动画…）。HTML 负责结构，CSS 负责外观，二者分离便于维护和复用。

任务 把标题改成 purple，。 
h1 { color: purple; }
p  { background: lightgray; }

如何把 CSS 加到页面（外链 / 内嵌 / 行内）

  • 外链（推荐）：<link rel="stylesheet" href="styles.css"> —— 复用性最好。
  • 内嵌：页面的 <style> 中写规则 —— 只影响当前页面。
  • 行内：元素的 style="" —— 只改某一个元素，不利于维护。
任务 调大 .note 的圆角，并给它加一条 lightgray 边框（编辑器里改 CSS）。
要求： 颜色用命名色（lightgray），不要用十六进制。 
.note{
  border: 1px solid lightgray;
  border-radius: 12px;
}

CSS 语法：选择器 + 声明块

一条规则由选择器和声明块组成：p { color: white; background: purple; }。

选择器（基础到进阶）

基础：通配（*）、标签（p）、类（.note）、ID（#title）、分组（h1, h2）。进阶：关系（后代/子代/兄弟）、伪类（:hover）、伪元素（::first-line）。

示例（只读）
/* 标签 */        p{ color: black }
/* 类 */          .note{ background: lightyellow }
/* ID */          #title{ color: purple }
/* 分组 */        h1, h2{ font-family:Arial, Helvetica, sans-serif }
/* 关系/伪类 */   .card p strong{ color: red }  a:hover{ text-decoration: underline }
小任务
  • 让 .note 段落背景变成 lightyellow。
  • 让 .card p strong 只在卡片内变 red。
  • 要求： 颜色仅用命名色。
.note { background: lightyellow; }
.card p strong { color: red; }

颜色与单位（命名色 + px/%/em/rem/vw/vh）

此处练习用命名色；单位用相对单位体会响应式。

文字与字体（font-family / size / weight / line-height / letter-spacing）

通过字体族、字号、行高、粗细、字距等让文字更易读。

间距（margin / padding）与简写

margin: 上 右 下 左;；两值/三值/四值的简写要掌握；padding 同理。

任务 用一条简写把 margin-top:20px; margin-bottom:12px; margin-left:8px; margin-right:8px; 合并。 
/* 上 | 左右 | 下 */
margin: 20px 8px 12px;

边框与背景（border / border-radius / background）

边框粗细、样式、颜色，圆角与背景色/图像都很常用。

盒模型（content + padding + border + margin）

推荐 box-sizing: border-box;，计算更直观。

显示与定位（display / position / z-index / overflow）

display：block / inline / inline-block / none；position：static / relative / absolute / fixed / sticky。

Flex 布局（现代首选的一维布局）

主轴方向（row/column）、对齐（justify-content / align-items）、换行（flex-wrap）、伸缩（flex）。

任务 把布局改成 column，并让中间那张卡片 flex: 2（更大）。 
.row { flex-direction: column; }
.row > div:nth-child(2) { flex: 2; }

级联（Cascade）与优先级（Specificity）

一般规则：行内 > #id > .class > 标签；后写覆盖先写；!important 最高（慎用）。

任务 删除示例中 style="color:orange" 的行内样式，再给 p 加 !important，比较谁赢。 
/* 删除行内后，ID 更高 → red */
p{ color: green }
.note{ color: blue }
#emph{ color: red }

/* 若再将 p 提升： */
p{ color: green !important }
/* 则 p 胜出（不建议滥用 !important） */

继承（Inheritance）

颜色/字体会继承；尺寸/边框/外边距等不继承。可用 inherit / initial / revert 控制。
